# CivicOne (Civisense)

A web-based platform for citizens to report and track civic issues like road damage, water supply, and electricity.

## Tech Stack
- **Frontend:** HTML5, CSS3 (Bootstrap 5)
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL

## Setup Instructions
1. Clone the repository: `git clone https://github.com/Vibhastalks/Civic-One.git`
2. Install dependencies: `npm install`
3. Set up your `.env` file with your PostgreSQL credentials.
4. Run the server: `node server.js`
